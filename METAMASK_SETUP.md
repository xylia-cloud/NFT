# MetaMask 配置详细指南

## 📱 第一步：安装MetaMask（如果还没有）

1. 访问 [MetaMask官网](https://metamask.io/)
2. 点击"下载"按钮
3. 选择你的浏览器（Chrome、Firefox、Edge等）
4. 按照提示安装扩展
5. 创建新钱包或导入现有钱包（首次使用）

## 🌐 第二步：添加本地Hardhat网络

### 方法一：通过界面添加（推荐）

1. **打开MetaMask扩展**
   - 点击浏览器右上角的MetaMask图标（🦊）
   - 如果被锁定，输入密码解锁

2. **进入网络设置**
   - 点击网络下拉菜单（通常在顶部，显示"以太坊主网络"等）
   - 滚动到底部，点击 **"添加网络"** 或 **"添加网络（手动）"**

3. **填写网络信息**
   在表单中填入以下信息：
   
   ```
   网络名称: Hardhat Local
   RPC URL: http://127.0.0.1:8545
   链ID: 1337
   货币符号: ETH
   区块浏览器URL: (留空)
   ```
   
   **注意**：
   - RPC URL 必须是 `http://127.0.0.1:8545`（不是 `localhost`）
   - 链ID 必须是 `1337`（数字）
   - 货币符号是 `ETH`

4. **保存网络**
   - 点击 **"保存"** 或 **"添加"** 按钮
   - 网络会自动切换到你刚添加的 "Hardhat Local" 网络

### 方法二：通过设置添加

1. 点击MetaMask右上角的 **账户图标**（圆形头像）
2. 选择 **"设置"** 或 **"Settings"**
3. 在左侧菜单选择 **"网络"** 或 **"Networks"**
4. 点击 **"添加网络"** 或 **"添加网络（手动）"**
5. 填写上述网络信息并保存

## 🔑 第三步：导入测试账户

### 获取测试账户私钥

在你的终端中（运行 `npm run node` 的窗口），你会看到类似这样的输出：

```
Account #0: 0xf39Fd6e51aad88F6F4ce6aB8827279cffFb92266 (10000 ETH)
Private Key: 0xac0974bec39a17e36ba4a6b4d238ff944bacb478cbed5efcae784d7bf4f2ff80
```

**复制任意一个账户的私钥**（从 `0x` 开始到结束的完整字符串）

### 导入账户到MetaMask

1. **打开MetaMask**
   - 点击浏览器右上角的MetaMask图标
   - 确保已切换到 "Hardhat Local" 网络

2. **进入导入界面**
   - 点击右上角的 **账户图标**（圆形头像）
   - 选择 **"导入账户"** 或 **"Import Account"**

3. **粘贴私钥**
   - 在输入框中粘贴你复制的私钥
   - 或者选择 **"私钥"** 选项，然后粘贴

4. **确认导入**
   - 点击 **"导入"** 或 **"Import"** 按钮
   - 账户会立即出现在你的账户列表中

5. **验证账户**
   - 你应该看到账户余额显示为 **10000 ETH**
   - 账户地址应该与终端中显示的地址一致

## ✅ 验证配置

### 检查网络连接

1. 确保MetaMask显示的网络是 **"Hardhat Local"**
2. 点击网络名称，应该能看到：
   - 网络名称: Hardhat Local
   - RPC URL: http://127.0.0.1:8545
   - 链ID: 1337

### 检查账户余额

1. 在主界面应该看到：
   - 账户地址（以 `0x` 开头）
   - 余额：**10000 ETH**（或接近这个数字）

### 测试连接

1. 打开你的DApp前端（`http://localhost:3000`）
2. 点击 **"连接钱包"** 按钮
3. MetaMask应该弹出连接请求
4. 点击 **"连接"** 或 **"Connect"**
5. 如果成功，前端应该显示你的账户地址

## 🐛 常见问题排查

### 问题1：无法连接到网络

**症状**：MetaMask显示"无法连接到网络"

**解决方法**：
- 确保Hardhat节点正在运行（`npm run node`）
- 检查RPC URL是否正确：`http://127.0.0.1:8545`
- 尝试重启MetaMask扩展
- 检查防火墙是否阻止了本地连接

### 问题2：余额显示为0

**症状**：导入账户后余额显示0 ETH

**解决方法**：
- 确保已切换到 "Hardhat Local" 网络
- 检查是否导入了正确的账户（地址应该匹配）
- 尝试刷新MetaMask（关闭并重新打开扩展）

### 问题3：无法导入账户

**症状**：粘贴私钥后无法导入

**解决方法**：
- 确保私钥完整（以 `0x` 开头，共66个字符）
- 检查是否有多余的空格或换行
- 尝试复制私钥时不要包含 "Private Key: " 前缀

### 问题4：交易失败

**症状**：在DApp中操作时交易失败

**解决方法**：
- 确保账户有足够的ETH（应该有10000 ETH）
- 确保已切换到正确的网络（Hardhat Local）
- 检查合约是否已正确部署
- 查看MetaMask中的错误信息

## 💡 提示

1. **使用测试账户**：这些账户的私钥是公开的，**绝对不要**在主网或其他真实网络中使用
2. **保存私钥**：开发时可以保存几个测试账户的私钥，方便切换使用
3. **重置账户**：如果需要重置，可以删除账户并重新导入
4. **多个账户**：可以导入多个测试账户，方便测试不同场景
